# Recharge Calculator 🔋📱

<div style="font-family: Arial, sans-serif; line-height: 1.6;">

<h2 style="color:#2d89ef;">📌 About</h2>
<p>
Recharge Calculator is a <b>personal-use fullstack application</b> built with <b>Next.js</b> and <b>MongoDB</b>.  
It is designed to help manage and track mobile/DTH recharge records efficiently.  
The system is secured for <b>only one admin login</b>, ensuring private access and usage.
</p>

<h2 style="color:#e81123;">⚡ Features</h2>
<ul>
  <li>🔐 <b>Secure Admin Login</b> – Only the admin can log in and access the system.</li>
  <li>📒 <b>Recharge Records</b> – Store details of recharges made for different persons.</li>
  <li>⏳ <b>Expiry Tracking</b> – Highlights recharges that are nearing expiry in the dashboard.</li>
  <li>📊 <b>Dashboard View</b> – A quick overview of all recharge activities.</li>
  <li>💾 <b>MongoDB Storage</b> – Data persistence with cloud database integration.</li>
</ul>

<h2 style="color:#107c10;">🚀 Tech Stack</h2>
<ul>
  <li><b>Frontend:</b> Next.js (React Framework)</li>
  <li><b>Backend:</b> Next.js API Routes</li>
  <li><b>Database:</b> MongoDB</li>
  <li><b>Hosting:</b> Vercel</li>
</ul>
